WDS quantitative analyses (new recipe)

Rev. 4/18/2024 LRM

Note: this recipe describes how to set up an uncalibrated WDS quantitative analysis recipe. If you want to make a calibrated recipe, do this first and then refer to the “Calibration” recipe to apply a calibration to the existing recipe that you create here.

Before you start

Make a plan that includes the following pieces of information:

  1. Which elements you want to analyze.
  2. If you plan to eventually do a calibration, which standards you will use.
  3. What reference materials you plan to use a “check samples” to make sure that your results are accurate. Some people just analyze their standards as unknowns, which is fine.
  4. What spectrometer configuration you plan to use (the software can help you with this).
  5. What beam power, count time, spot size you plan to use (the “X-ray measure” tool can help you with this).

To get started, click the “Quick” button in the “PC-EPMA” program on the right side monitor, select the “Common recipe” button, and double click the “New Quantitative Analysis” recipe on the list to the right.

Gray/Green menus: File path/beam conditions

  1. Set the “Path” field to the main folder on the E: drive where the data will be stored
  2. Set the “Project Name” field according to the analyses being done (this creates a subfolder in the “Path” directory when data are generated)
  3. Input/read the beam conditions (voltage/current). Note that the checkboxes control whether the instrument will set the beam conditions automatically or use the existing conditions.

Purple section: Quantitative analysis condition

  1. Check “detail” box to open the detailed view of the settings
  2. Check to confirm that these settings are used:
    • Analysis in asynchronous mode
    • Peak “none” – don’t do peak searches during the analyses unless you have a good reason to do so (e.g. to correct for temperature fluctuations in the room).
    • Background “Always” (so the background will be analyzed every time)
    • Read probe current “Always” (so it will read the probe current before each analysis)

Orange section: Analysis element condition

  1. Check “detail” box to open the detailed view of the settings
  2. Click the periodic table button (“element” tab should be selected instead of the “search” tab)
  3. Click the elements which will be analyzed using this recipe.
  4. For each element:
    • Default PHA settings
    • Peak search: #1 (only one narrow search should be necessary to find peaks. The spectrometer reproducibility is very good.)
    • Set the count time and background point spacing (e.g. 5mm means to use an L-value 5 mm below/above the peak position as background)

Note: example spectra stored in EPMA Data > JEOL Spectra can be used to check for interferences

When you’re all done

  1. Save the recipe using a descriptive name including the date, your name, and what the recipe will be used to analyze.
  2. Yellow section: analysis position condition: Save this for last: add points and perform analyses according to the “existing recipe” SOP now that you have created an extant recipe to use!
